Combat SDK Help

Константы

Для параметров $whom
$myselfВы
$opponentВаш оппонент
$friendsВы и союзники
$enemiesВсе враги
Для параметров $where
$current_creatureВыбранное существо
$all_creaturesВсе существа
$creature_slot_1Существо в слоте 1
$creature_slot_2Существо в слоте 2
$creature_slot_3Существо в слоте 3
Для параметров $attrib
$creature_attackАтака существа
$creature_defenceЗащита существа

Функции, возвращающие информацию о сражении

get_turn_number( );
Возвращает номер текущего хода.

Функции, воздействующие на персонажей

kill( $whom );
Убивает указанную в качестве параметра $whom цель.

damage( $whom, $value );
Наносит цели $whom $value урона.

heal( $whom, $value );
Восстанавливает цели $whom $value жизней.

alter_attrib( $whom, $attribute_id, $value );
Изменяет цели $whom значение аттрибута $attribute_id на $value.

set_attrib( $whom, $attribute_id, $value );
Устанавливает цели $whom значение аттрибута $attribute_id в $value.

Функции, возвращающие информацию о персонажах

get_attrib_value( $whom, $attribute_id );
Возвращает значение аттрибута $attribute_id у цели $whom.
$whom не может быть $friends или $enemies.

Функции, воздействующие на существ

summon( $creature_id, $where = $current_creature );
Призывает в выбранный слот существо $creature_id.

kill_creature( $whom, $where );
Убивает свое существо или существо оппонента в слоте $where, в зависимости от параметра $whom.
$whom не может быть $friends или $enemies.

alter_creature_attrib( $whom, $where, $attrib, $value );
Увеличивает выбранному существу у себя ($whom == $myself) или у оппонента ($whom == $opponent) значение аттрибута $attrib ($creature_attack или $creature_defence) на $value.
$whom не может быть $friends или $enemies.

set_creature_attrib( $whom, $where, $attrib, $value );
Устанавливает выбранному существу у себя ($whom == $myself) или у оппонента ($whom == $opponent) значение аттрибута $attrib ($creature_attack или $creature_defence) в $value.
$whom не может быть $friends или $enemies.

Функции, возвращающие информацию о существах

get_creature_attrib_value( $whom, $where, $attrib );
Возвращает значение аттрибута $attrib у существа $where у персонажа $whom.
$whom не может быть $friends или $enemies.
$where не может быть $all_creatures.

has_creature_in_slot( $whom, $where );
Возвращает айди существа существа в слоте $where у персонажа $whom или false если существо в слоте не призывано.
$whom не может быть $friends или $enemies.
$where не может быть $all_creatures.
